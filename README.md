## Data

This is data from: http://www.californiacityfinance.com/#SPENDING

## Contributing

Wanna help?

```
git clone git@github.com:simonv3/police-data.git
cd police-data
yarn start
```

## Deploying to GitHub

To deploy to GitHub you need to run

```
yarn deploy
```

in your local repository.
